# oficina-ifro-busca-cep
App React Native com Expo para busca de CEP

# Requimentos:

- Node.js (https://nodejs.org)
- Expo cli (npm i -g expo-cli)
- VS Code

# Como executar o projeto

- Entre na pasta raiz do projeto
- Execute o comando "npm start"
